# Lotus House — Website

Public deploy target for the Lotus House brand identity project — a children & adolescent psychotherapy practice.

Currently used as a **multi-device test harness**: brand work in progress (typography sheets, composed layouts) is published here via GitHub Pages so it can be reviewed on phone, tablet, and desktop during the design process. Will eventually become the launched website.

## How this gets updated

Files here are generated by a deploy script in the (private) brand source repo. Don't edit directly — they're overwritten on each deploy.
